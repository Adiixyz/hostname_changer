# CHANGE HOSTNAME UBUNTU
A tool to change hostname

# How To Install?
Follow This
```cmd
apt install git
apt install sudo
git clone https://github.com/Adiixyz/hostname_changer
cd hostname_changer
```

# Please give a permission
`chmod +x hostchange.sh`

# Change IT!
`./hostchange.sh`
